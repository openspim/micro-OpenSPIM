TIME-POINTS

Adding multiple time-points

In order to set up a time-lapse recording of a given time span, press the “Add New TP” button and define the number of the intended time-points (TP) within the newly created entry below and choose the desired interval between time-points.

Adding multiple timepoints with different intervals and/or breaks

It is possible to add several time-point entries with the “Add New TP” button, which can have different intervals. Additionally, breaks can be added by pressing the “Add New Pause” button and specifying its timespan.

Deleting created entries

Select any previously created position from the list and then press the “Delete position” button to remove the entry.

Deleting created time-points

Select any previously created time-point from the list and then press the “Delete TP” button to remove the selected entry from the list.
